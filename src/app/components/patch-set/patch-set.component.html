SetValue:

  • SetValue is used to set values for all controls in the form group.
  • The provided object must contain values for all controls in the form group.
  • If any control is missing in the provided object or the values are not valid, it will throw an error.

PatchValue:

  • PatchValue is used to set values for specific controls in the form group.
  • The provided object can contain values for only a subset of the form controls.
  • If a control is not present in the provided object, its value remains unchanged.
  • It does not throw an error if controls are missing.
